# AI Resume Architect

AI Resume Architect is a professional, AI-driven resume builder developed for **Acadeno Technologies**. It leverages advanced Natural Language Processing (NLP) to analyze job descriptions and automatically fetch your best projects from GitHub, transforming them into high-impact professional achievements.

##  Features

- **GitHub Integration**: Automatically fetches and reflects your real-world coding contributions.
- **NLP-Driven Skill Matching**: Analyzes job descriptions to ensure your resume hits high ATS (Applicant Tracking System) scores.
- **Executive Summary Generation**: Crafts punchy, professional summaries tailored to your profile.
- **Live Preview & Edit**: Real-time interactive editing before generating your final document.
- **Professional PDF Export**: Produces high-quality, ATS-optimized executive resumes.

##  Tech Stack

- **Backend**: Python (Flask)
- **Frontend**: HTML5, Vanilla CSS, JavaScript
- **AI/NLP**: Advanced keyword extraction and scoring algorithms
- **PDF Engine**: Professional LaTeX-based PDF generation

##  Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd AI_Resume_Builder/backend
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   python app.py
   ```
   The application will be available at `http://127.0.0.1:5000`.
